# Loonaro-KVM

**Polymorphic VM orchestration with KVM-VMI introspection.**

## Structure

| Directory | Purpose |
|-----------|---------|
| `kvm-vmi/` | (Submodule) Patched QEMU/KVM for introspection |
| `loonaro/` | VM templates, scripts, configs |

## Getting Started

1. **Clone**: `git clone --recursive git@github.com:416rehman/loonaro-kvm.git`
2. **Build Engine**: Follow [KVM-VMI Setup](https://kvm-vmi.github.io/kvm-vmi/master/setup.html)
3. **Use Loonaro**: See **[loonaro/README.md](loonaro/README.md)**
